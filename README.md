## Installation

```bash
$ npm install
```

## Running the app

Start the server
```bash
$ npm run start:dev
```

In another terminal:
First seed the data to create two sample providers and clients

```bash
$ npm run seed
```
You can check the available providers and clients by running:

```bash
$ npm run query
```


You can check the swagger documentation at:
```bash
$ http://localhost:3000/api
```
## Test

```bash
$ npm run test
```
